## コンソール使用ガイド

**Data & Analytics > Data Lake Storage > コンソール使用ガイド**

Data Lake Storageサービスのコンソールは、以下の機能を提供します。

* バケットの照会と管理
* 認証情報の管理と照会
* オブジェクトの一覧及び詳細情報の照会

## バケットの照会

### バケット一覧の照会

作成したバケット一覧は、Data Lake Storageサービスのコンソールで確認できます。

* 名前の左側にあるチェックボックスを使用して、バケットを選択できます。
* 一覧の右上にある**更新**をクリックして一覧を更新できます。

| 項目 | 説明 |
| --- | --- |
| 名前 | バケット名 |
| オブジェクト数 | バケットに保存されたオブジェクトの数 |
| サイズ | バケットに保存されたオブジェクトの合計サイズ |

### バケットの検索

バケット一覧の左上にある入力エリアと虫眼鏡ボタンを使用して、入力した文字列を含む名前のバケットを検索できます。

| 項目 | 必須 | 説明 | 制限 |
| --- | --- | --- | --- |
| 検索ワード | O | 検索するバケットの名前 | 英小文字と数字、記号('`.`'、'`-`')を使用できます。 |

### バケットの詳細照会

バケット一覧でバケット名以外の領域をクリックすると、バケットの詳細情報を照会できます。

| 項目 | 説明 |
| --- | --- |
| バケット名 | バケットの名前 |
| 説明 | バケット作成時に登録した説明 |
| オブジェクト数 | バケットに保存されたオブジェクトの数 |
| サイズ | バケットに保存されたオブジェクトの合計サイズ |
| 作成日時 | バケットの作成時刻 |
| 最終更新日時 | バケットが最後に更新された時刻 |

## バケットの管理

### バケット作成

バケット一覧の上部にある**バケット作成**をクリックしてバケットを作成できます。バケット作成時に必要な入力値は以下の表のとおりです。

!!! tip 「ポイント」
    Data Lake Storageサービスのバケット名はリージョン内で一意です。

| 項目 | 必須 | 説明 | 制限 |
| --- | --- | --- | --- |
| 名前 | O | バケットの名前 | 3 ～ 63文字の長さにする必要があります。<br>英小文字と数字、記号('`.`'、'`-`')を使用できます。<br>名前の先頭と末尾には英小文字または数字を使用する必要があります。<br>連続する `.`(`..`)、IPv4パターン、一部のプレフィックス及びサフィックスは使用できません。<br>使用できないプレフィックス：`xn--`、`sthree-`、`amzn-s3-demo-`<br>使用できないサフィックス：`-s3alias`、`--ol-s3`、`.mrap`、`--x-s3`、`--table-s3` |
| 説明 | X | バケットの説明 | 100文字以内の長さにする必要があります。 |

### バケット削除

バケット一覧のチェックボックスを使用してバケットを選択した後、一覧の上部にある**削除**をクリックしてバケットを削除できます。

!!! danger "注意"
    削除の対象は、オブジェクトが全て削除された空のバケットである必要があります。バケットが空でない場合は、オブジェクト削除APIやバケットを空にする機能を使用して、事前にバケットを空にしておく必要があります。

バケットの削除は元に戻せません。

### バケットを空にする

バケット一覧のチェックボックスを使用してバケットを選択した後、一覧の上部にある**バケットを空にする**をクリックして、バケット内の全てのオブジェクトの削除をリクエストできます。

| 項目 | 必須 | 説明 | 制限 |
| --- | --- | --- | --- |
| 名前 | O | 空にするバケットの名前 |  |

!!! danger "注意"
    オブジェクト数に応じて時間がかかる場合があります。

バケットを空にする処理は、途中でキャンセルしたり元に戻したりすることはできません。

## 認証情報の管理

### 認証情報の照会

バケット一覧照会ページの上部にある**S3 API 認証情報**をクリックすると、認証情報の管理ポップアップが表示されます。ポップアップ内で、発行された認証情報の一覧を確認できます。

| 項目 | 説明 |
| --- | --- |
| Access Key | 認証情報のAccess Key |
| 説明 | 認証情報の発行時に登録した説明 |
| 作成日時 | 認証情報の作成日時 |
| 最終使用日時 | 認証情報の最終使用日時 |
| 管理 | 修正及び削除機能 |

### 認証情報の発行

S3 API認証情報ポップアップの上部にある**S3 API 認証情報登録**をクリックして、新しい認証情報を発行できます。認証情報の発行時に、使用目的などの説明を追加できます。

| 項目 | 必須 | 説明 | 制限 |
| --- | --- | --- | --- |
| 説明 | X | 認証情報の用途などの説明 | 100文字以内 |

!!! tip 「ポイント」
    ユーザーアカウントごとに、1プロジェクトあたりS3 API認証情報を最大3つまで発行できます。

!!! danger "注意"
    APIを使用する際、ユーザー認証のために認証情報を使用します。流出した場合、他人がストレージの機能やオブジェクトにアクセスできる可能性があります。必ず安全な場所に保管し、他人と共有しないでください。
    認証情報を発行した直後に、Access KeyとSecret Keyが表示されます。Secret Keyは発行直後の1回のみ確認可能です。

### 認証情報の修正

S3 API認証情報一覧の管理領域にある**修正**をクリックして、認証情報の説明を修正できます。

| 項目 | 必須 | 説明 | 制限 |
| --- | --- | --- | --- |
| 説明 | X | 認証情報の用途などの説明 | 100文字以内 |

### 認証情報の削除

S3 API認証情報一覧の管理領域にある**削除**をクリックして、認証情報を削除できます。このとき確認のポップアップが表示されます。

!!! danger "注意"
    削除した認証情報ではAPI認証を行うことができません。また、削除した認証情報は復元できません。

## オブジェクトの照会

### オブジェクト一覧の照会

バケット一覧でバケット名をクリックすると、バケット内のオブジェクト一覧を照会できます。

| 項目 | 説明 |
| --- | --- |
| 名前 | オブジェクト名 |
| サイズ | オブジェクトのサイズ |
| 最終更新日時 | オブジェクトをアップロードした時刻 |

### オブジェクトの詳細照会

オブジェクト一覧で該当するオブジェクトをクリックすると、詳細情報を照会できます。

| 項目 | 説明 |
| --- | --- |
| オブジェクト名 | オブジェクトの名前 |
| サイズ | オブジェクトのサイズ |
| ETag | オブジェクトのバージョン情報 |
| 最終更新日時 | オブジェクトをアップロードした時刻 |
| ストレージクラス | オブジェクトのストレージクラス |
